I have a hard time using JSON for my games and sometimes when I use it for my job, I just turn back to MySQL and although it's faster for development, I would really just like a simple text-based database that can also work in tables. I think there could be more robust application that replicate this but I figured, I am a software dev, why not create my own for my own requirements, and so voila!
